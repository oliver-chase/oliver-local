# oliver-local

**Repository:** `oliver-chase/oliver-local` (remote)

## What this repo is

This is a **workflow and orchestration workspace** (Codex/Claude runtime), not an application codebase.

It contains:
- `CLAUDE.md` and `README.md` operational instructions
- shared workflows (`shared/workflows/*`)
- skill definitions (`skills/*`)
- plugin/tooling configuration
- command hooks and local agent/project documentation

## What it is not

It is **not** where feature/product code for `oliver-app`, `v-two-sdr`, `fallow`, or `tesknota` lives.

## Where product repos live

- `~/projects/oliver-app`
- `~/projects/v-two-sdr`
- `~/projects/fallow`
- `~/projects/tesknota`

## Canonical local path relationship

- Working tree root: `~/oliver-local`
- Canonical command alias: `~/oliver-local`
